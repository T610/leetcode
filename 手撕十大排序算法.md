# 冒泡排序

```python
def swap(i,j):
            temp = num[i]
            num[i] = num[j]
            num[j] = temp
        for i in range(len(num)):      
            for j in range(len(num)-i-1):  
                if num[j] > num[j+1] :   
                    swap(j,j+1)          
```
